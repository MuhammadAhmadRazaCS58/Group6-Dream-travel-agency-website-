### **AICTL Web Development Capstone**

Welcome to the official repository for the **Dream Travel Agency** website. This project serves as the final capstone for our AICTL course, demonstrating a comprehensive understanding of modern front-end design principles using foundational web technologies.

## Project Overview

The **Dream Travel Agency** web platform is a multi-page, fully responsive site designed to provide a seamless user experience for travelers. The goal was to create a visually appealing interface that balances aesthetic travel imagery with functional navigation and clear calls to action.
---
### **Key Features**

* **Dynamic Landing Page:** A high-impact hero section designed to engage users immediately.
* **Curated Gallery:** A dedicated space showcasing global destinations with a focus on visual storytelling.
* **Full Responsiveness:** Optimized for viewing across desktops, tablets, and mobile devices.
